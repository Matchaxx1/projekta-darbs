AquaGO!

Aquarium Steps ir 2D Android mobilā spēle, kuras mērķis ir motivēt bērnus un jauniešus būt fiziski aktīvākiem. Lietotāja reālajā dzīvē veiktie soļi tiek pārvērsti virtuālajās monētās, kuras iespējams izmantot, lai iegādātos zivis un veidotu savu akvāriju.
Spēle apvieno fizisko aktivitāti ar vienkāršu un motivējošu spēles mehāniku.

Galvenās funkcijas:

Soļu skaitīšana</br>
Monētu aprēķināšana no soļiem</br>
Zivju iegāde veikalā</br>
Zivju attēlošana akvārijā</br>
Lietotāja datu saglabāšana</br>
</br>
Izmantotās tehnoloģijas:</br>
Unity (C#)</br>
Datubāze (SQLite / Firebase)</br>
GitHub versiju kontrolei</br>
</br>
</br>
Instalēšana uz telefona:</br>
1.Lejuplādēt APK failu no Release sadaļas</br>
2. Instalēt Android ierīcē</br>
3. Atļauj instalēšanu no nezināmiem avotiem, ja nepieciešami</br>
</br>
Ja projekts tiek atvērts Unity vidē obligāti jāielādē un jāimportē Firebase Dependencies.
